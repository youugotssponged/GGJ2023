# GGJ2023
Global Game Jam 2023 - Rooty Tooty Shooty

A neat-o tower-defence game all about collecting roots and protecting "The Ancient one" a tree that has been your home for many centuries. That is UNTIL the day that the orks arrived destroying everything and everyone. Will you take up arms and command the land to stop these brutes from taking your home?
